PROJET ENTREPRISE : PHARMA PILOT

Pharma Post, une société de livraison dédiée à la santé, nous a contacté afin de créer une application qui aidera à anticiper une potentielle prochaine pandémie de la COVID.

Le besoin est de pouvoir restituer un nombre de doses nécessaire par département.

Source : Explorateur de données : data.gouv.fr

Afin de répondre à ce besoin, nous devons créer une application qui affiche les doses de vaccins, par département, selon le numéro de semaine à n-3.

En tant que développeur python, vous devrez prendre en charge ce besoin. Votre mission sera de :

  • Créer une application et la publier sur Github
  • Stocker toutes les données provenant de data.gouv dans une base de données dans une modélisation en étoile.
  • Créer un script qui alimente la table ODS et les tables de l’entrepôt de données.
  • Restituer les données de chaque table via une API. Seules les requêtes GET sont autorisées via un Token.
  • Rédiger des tests cases pour chaque table du DWH. Au moins 5 tests devront être rédigés.
  • Publier dans discord (via api et dans la conversation Python 2), le nombre de vaccin nécessaire pour le Puy de dôme, selon la semaine de publication du message. Votre message devra comporter le lien de votre répertoire GITHUB.
